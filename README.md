# GitHub Trend Tracker

> GitHubリポジトリのトレンドを定量指標で可視化するWebサービス

## 特徴

- 📊 **言語別トレンドランキング** - 7日/30日の増加率・増加数でソート可能
- 📈 **時系列グラフ** - リポジトリのスター数推移を可視化
- 🔄 **自動データ収集** - GitHub Actionsによる日次バッチ処理
- ⚡ **高速・低コスト** - Cloudflare Workers/D1による無料運用

## クイックスタート

詳細は [セットアップガイド](./docs/基本/セットアップガイド.md) を参照してください。

```bash
# 依存関係のインストール
npm install

# Cloudflareログイン
cd apps/backend && npx wrangler login

# 開発サーバー起動
npm run dev:backend  # API: http://localhost:8787
npm run dev:frontend # Web: http://localhost:4321
```

## 技術スタック

| レイヤー     | 技術                      |
| ------------ | ------------------------- |
| **Frontend** | Astro + React             |
| **Backend**  | Cloudflare Workers (Hono) |
| **Database** | Cloudflare D1 (SQLite)    |
| **ORM**      | Drizzle ORM               |

## コスト

完全無料で運用可能（Cloudflare Free枠内）：

- **D1**: 5GB、500万読み取り/日、10万書き込み/日
- **Workers**: 10万リクエスト/日
- **Pages**: 無制限リクエスト
