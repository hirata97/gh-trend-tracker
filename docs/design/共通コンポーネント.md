# 共通コンポーネント

再利用可能なUIコンポーネントを定義する。

## コンポーネント一覧

| ID | 分類 | ファイル名 | 概要 |
|----|------|-----------|------|
| COM-001 | リストアイテム | RepositoryListItem | リポジトリ一覧表示用カード/行 |
| COM-002 | レイアウト | GlobalHeader | サイト全体のヘッダーナビゲーション |
| COM-003 | フィルタリング | LanguageFilterDropdown | 言語選択ドロップダウン |
| COM-004 | ナビゲーション | Pagination | ページ送りUI |
| COM-005 | UI要素 | PrimaryButton | 主要アクションボタン |

---

## COM-001: RepositoryListItem

リポジトリの一覧表示に使用されるカード/行コンポーネント。

- **分類**: リストアイテム
- **ファイル名**: `RepositoryListItem.tsx`

### Props

```typescript
interface RepositoryListItemProps {
  repo: RepositoryObject;
  showMetrics: boolean;
}
```

### 表示内容

- リポジトリ名（full_name）
- 概要（description）
- スター数（stargazers_count）
- 7日間増加数（stars_7d_increase）- showMetricsがtrueの場合
- 30日間増加数（stars_30d_increase）- showMetricsがtrueの場合

### 使用画面

- [scr-001](./画面一覧.md#scr-001-トレンド一覧画面-日次): トレンド一覧画面
- [scr-003](./画面一覧.md#scr-003-週別トレンドランキング画面): 週別トレンドランキング画面
- [scr-004](./画面一覧.md#scr-004-検索結果一覧画面): 検索結果一覧画面
- [scr-010](./画面一覧.md#scr-010-お気に入り一覧画面): お気に入り一覧画面

## COM-002: GlobalHeader

- **分類**: レイアウト
- **ファイル名**: `GlobalHeader.tsx`

### Props

```typescript
interface GlobalHeaderProps {
  snapshotDate: string;
  isAuthenticated: boolean;
}
```

### 表示内容

- サイトロゴ
- 主要ナビゲーションリンク
- データスナップショット日時（[fun-014](../requirements/機能要件一覧.md#fun-014-データスナップショット日時表示)）

### 使用画面

- 全画面で使用
