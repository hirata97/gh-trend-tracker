## scr-001: トレンド一覧画面 (日次)

GitHubリポジトリの言語別日次トレンドランキング（Top 100）を表示するメインページ。

- **分類**: 閲覧/一覧
- **利用者**: エンジニア, 技術ブロガー

### 構成要素

| 要素             | 関連機能             | 説明                           |
| ---------------- | -------------------- | ------------------------------ |
| ヘッダー         | fun-014              | データスナップショット日時表示 |
| 言語選択UI       | fun-006              | 言語フィルタドロップダウン     |
| 検索入力フォーム | fun-015              | リポジトリ名検索               |
| ソート基準選択UI | fun-008/009/010      | 7日/30日増加数/率、総スター数  |
| リポジトリリスト | fun-003, fun-004/005 | 基本情報と増加数表示           |
| ページネーション | fun-053              | ページ切り替え                 |
| レスポンシブUI   | fun-029              | デバイス対応                   |

### 操作手順

1. 言語を選択してフィルタリング (fun-007)
2. 7日間/30日間の増加数または増加率でソートを実行
3. リポジトリを選択して詳細へ遷移 (fun-011)
4. リポジトリ名を検索窓に入力 (fun-016)
5. ページ切り替え (fun-053)

## scr-002: リポジトリ詳細画面

特定のリポジトリの基本情報と過去90日間のスター推移をグラフで表示する。

- **分類**: 閲覧/詳細
- **利用者**: エンジニア, 技術選定者

### 構成要素

| 要素                  | 関連機能         | 説明                         |
| --------------------- | ---------------- | ---------------------------- |
| リポジトリ基本情報    | fun-002, fun-003 | 名前、概要、スター、フォーク |
| トピックス情報        | fun-048          | GitHubトピックスタグ         |
| スター推移グラフ      | fun-013          | 90日間の折れ線グラフ         |
| グラフツールチップ    | fun-045          | マウスオーバーで詳細表示     |
| AI要約閲覧/生成ボタン | -                | 要約画面への導線             |
| お気に入り登録ボタン  | fun-047          | ローカルストレージ保存       |

### 操作手順

1. グラフにマウスオーバーし詳細データを確認 (fun-045)
2. AI要約閲覧ボタンをクリックし、要約画面に遷移/モーダル表示
3. お気に入り登録/解除 (fun-047)
